
**Want AI-generated words?** Get a free API key from [Google AI Studio](https://makersuite.google.com/app/apikey) (takes 2 minutes).

## Step 3: Run the Game

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## ğŸ› Troubleshooting

**Game not loading?**
- Clear browser cache
- Try incognito/private mode
- Check console for errors (F12)

**AI words not working?**
- Verify your API key is correct in `.env.local`
- Check if you have internet connection
- The game will automatically fall back to offline words

**Players can see each other's roles?**
- Make sure each player hides the screen before passing
- Use the "Hide & Pass Device" button
- Consider using a privacy screen protector
